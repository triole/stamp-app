Anforderung

So weit möglich werden alle Daten und Begleitmaterialien, die im Projekt generiert wurden, Dritten über ein Repositorium oder Forschungsdatenzentrum zur Nachnutzung zur Verfügung gestellt. Dabei gilt, dass die Daten und Begleitmaterialien inhaltlich so umfangreich, rechtlich und forschungsethisch so offen und zeitlich im Projektverlauf so früh wie möglich verfügbar gemacht werden.

Erläuterung zur Anforderung

Die Projektbeteiligten machen im Projekt generierte Daten und Begleitmaterialien Dritten, so weit möglich, über ein geeignetes Repositorium oder Forschungsdatenzentrum zur Nachnutzung verfügbar. Dabei gilt, dass Daten und Begleitmaterialien die verfügbar gemacht werden können, (1) so zeitnah wie möglich, spätestens aber zum Projektende, (2) inhaltlich so umfangreich wie möglich, d. h. mit dem größtmöglichen (Nach-)Nutzungspotential und (3) so offen wie möglich, d. h. mit den geringstmöglichen Nutzungsbeschränkungen, verfügbar gemacht werden. Die Projektbeteiligten stellen, ggf. in Kooperation mit dem Repositorium bzw. Forschungsdatenzentrum, sicher, dass bei der Verfügbarmachung nicht gegen geltendes Recht, wie etwa den Datenschutz oder das Urheberecht, forschungsethische Grundsätze oder sonstige Auflagen (Dritter) verstoßen wird. Sie achten darauf, dass die Daten und Begleitmaterialien durch das Repositorium bzw. Forschungsdatenzentrum mit einem persistenten Identifikator versehen und in einem digitalen Datenkatalog registriert werden. Die Daten und Begleitmaterialien werden so frühzeitig für eine möglichst breite Nachnutzung verfügbar, auffindbar und zitierbar. Um die Verfügbarkeit der Daten und Begleitmaterialien sicherzustellen,


  • Aa entwickeln die Projektbeteiligten bereits in der Projektplanungsphase ein Konzept zur Verfügbarmachung der Daten und Begleitmaterialen,
  • Ba bereiten die Projektbeteiligten die Daten und Begleitmaterialien so zeitnah im Projektverlauf wie möglich auf die Verfügbarmachung durch ein Repositorium oder ein Forschungsdatenzentrum vor und
  • Ca machen die Projektbeteiligten die Daten und Begleitmaterialien Dritten zur Nachnutzung verfügbar.
